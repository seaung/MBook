##### Lua中的数据类型
Lua语言提供了六种基本的数据类型


| 数据类型 | 说明 |
| :-----: | :---:|
| nil     | 表示不存在的空对象或无效值   |
| boolean | 布尔类型，值为false或true   |
| number  | 表示双精度类型的实浮点数     |
| string  | 由一对双引号或单引号来表示   |
| function| 由C或lua编写的函数          |
| userdata| 表示任意存储在变量的C数据结构|
| thread  | 表示执行独立线路,用于执行协程|
| table   | 一组"关联数组"              |
